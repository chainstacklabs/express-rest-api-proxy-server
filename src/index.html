This webpage uses the Ethescan API to retrieve and display the latest block on the Ethereum network.

The idea of this app is to teach you how to properly protect your API keys!

Find the repository on GitHub

Inspecting the source code, you will notice that you won't find any API key in the front end; you will find instead a connection to a back-end proxy server where the API keys are safely stored. This is a simple way to protect keys used to query REST APIs.

Click the button to find out the latest block on the Ethereum blockchain!
